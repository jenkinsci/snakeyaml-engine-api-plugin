# SnakeYAML Engine API Plugin

This plugin exposes the [SnakeYAML Engine](https://bitbucket.org/snakeyaml/snakeyaml-engine) API to Jenkins plugins.

## Changelog

* See [GitHub Releases](https://github.com/jenkinsci/snakeyaml-engine-api-plugin/releases) for recent versions

## Developer documentation

See the [Jenkins Developer Documentation](https://jenkins.io/doc/developer/book/).

Cheatsheet:

* To build the plugin locally: `mvn clean verify`
* To test in a local Jenkins instance: `mvn hpi:run`
